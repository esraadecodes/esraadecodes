🚀 Featured Work

RAG System for Research Literature — Retrieval-augmented generation pipeline over technical research papers, built with hybrid retrieval (semantic + keyword) and evaluated using RAGAS (faithfulness, context precision/recall). Applies domain knowledge from computational neuroscience to a real information-access problem.

Linking RNN Structure to Function — Replication of neural manifold analysis (Feulner & Clopath, 2021) on RNN-based BCI models for center-out reaching tasks, using PCA/UMAP/Isomap to study how learning reorganizes network representations.

Comparing Networks (Neuromatch Academy) — Dynamical and causal comparison of rate-based RNNs vs. spiking neural networks (SNNs) for context-dependent decision-making, applying fixed-point analysis to characterize how each architecture solves the same task differently.
